# Protfolio WebSite
프론트엔드 신입 개발자 김태우의 포트폴리오입니다. <br/>
React를 이용해 만든 포트폴리오입니다. <br/>
포트폴리오 사이트인 만큼 기능적 요소보다는 UI/UX와 디자인에 더욱 신경써서 진행했습니다. 반응형 웹으로 만들어 대부분의 환경에서 사용가능합니다. <br/>
